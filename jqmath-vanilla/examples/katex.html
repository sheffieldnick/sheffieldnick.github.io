KaTeX is a JavaScript module that makes it easy to put formatted mathematical expressions in web pages. For example, the html source:

If \(ax^2+bx+c=0\) with \(a≠0\), then: $$x=\frac{-b±\sqrt{b^2-4ac}}{2a}$$

produces the following:

If \(ax^2+bx+c=0\) with \(a≠0\), then: $$x=\frac{-b±\sqrt{b^2-4ac}}{2a}$$

KaTeX uses TeX syntax:

  • \( and \), to delimit inline mathematics
  • $$, or \[ and \], to delimit block or “display” mathematics
  • ^ for superscript
  • _ for subscript
  • { and } for grouping
  • \overset for overscript
  • \underset for underscript
  • \ or ` to quote the next character (turning off its special meaning), start a macro or multi-letter name
  • \mathrm to make a name non-italic
  • \bf, \it, or \bm for a bold, italic, or bold-italic variable
  • \frak for a Fraktur letter
  • \overline for an overbar
  • \begin for a matrix, table, or simply tabular alignment - separate entries with & and rows with //
  • \text for plain text inside mathematics - specify the text inside {}
  • spaces, tabs and newlines are ignored - KaTeX formats the mathematics
  • you can use \$ for a real \$, or \\ for a \ if necessary

KaTeX is mediumweight, fast and complete. It is standards-based and cross-browser, using simple HTML and CSS for visual rendering and includes MathML for accessibility. Thus pages load quickly, and expressions can be resized by the user, or easily passed to screen reading, graphing, or computer algebra software.
